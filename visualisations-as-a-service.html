Visualisations as a Service

Inspired by my colleague Martin Costello's cool (and I presume tongue-in-cheek parody of everything-as-a-service these days), Advent of Code as a Service, I think I will create Advent of Code Visualisations as a Service.

Idea being: for the visualisations I'm proud of (i.e. the MonoGame ones), people can upload their input and get a video visualisation of the solution.

Not quite sure how I'll run MonoGame headless on a server (or better, in a Lambda) to return an mp4 (or suchlike), but I love a challenge.

Depending on my boredness/tiredness/hungoverness state on the weekend, you may see something soon!
